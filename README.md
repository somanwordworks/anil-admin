
# Anil Admin Panel - Starter Scaffold

This is a starter Next.js admin panel scaffold for admin.yourdomain.com.
It includes minimal pages, API routes, and utilities to connect to Neon Postgres.

## Quick start (local)

1. Copy `.env.example` to `.env.local` and fill DATABASE_URL and JWT_SECRET.
2. npm install
3. npm run dev
4. Open http://localhost:3000/login

## Notes
- This is a scaffold. Add styling, validation and production configuration as needed.
- Uses pg for Postgres connection and jsonwebtoken for auth.
